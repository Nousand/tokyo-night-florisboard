# Tokyo-Night-Florisboard
A passable port of the popular Tokyo Night theme to Florisboard


## Installation
### Via Addons Store
0. Navigate to the [Florisboard Addon Store](https://beta.addons.florisboard.org/)
0. Search for "Tokyo Night"
0. Press "Download"
0. Import The installation to Florisboard

### Via Source
0. Download the raw "tokyonight.flex" file
0. Navigate to "Addons & Extensions" in Florisboard settings
0. Press the "Import" button
0. Upload the "tokyonight.flex" to it
